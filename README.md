# Katsagra Residences — Interactive 3D Model

Interactive 3D massing model of three Cycladic residences with basements and pools at
Katsagra, Naxos, built from the February 2026 architectural permit set.

**Live model:** https://spyroaj95.github.io/katsagra-3d/

- Drag to orbit, scroll to zoom, click a house for levels and areas
- Viewpoint presets, sun-path shadow slider, layer toggles
- "Exploded floors" reveals basements and room-by-room interior labels
- Deep links: `#explode&view=aerial|road|pools|south&sun=0..100`

**Sharing with view tracking:** append `?v=name` to the link
(e.g. `…/?v=laoura`) — opens are counted per name, anonymously, on
[stats.html](https://spyroaj95.github.io/katsagra-3d/stats.html).

Single self-contained HTML file (Three.js inlined) — works offline if downloaded.
